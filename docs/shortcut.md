## 快捷键
### 说明
- 主力机为 Hackintosh，原机键盘为 windows 布局，对左下键盘布局进行了更改，由`crtl | fn | windows | alt` 更改为 `cmd | fn | crtl | alt`
- 概念
  - workspace:
  - editor：一个标签页，编辑单个文件
  - editor group
## 键位
- 预览：cmd+k,v 或 cmd+shift+v
- 视图(切换时，必须聚焦某一窗口上)
  - workspace
    - 切换已经打开的 workspace：crtl+w 
  - editor(同一 editor group 中的 editor)
    - 切换-左右：cmd+shift+ []
    - 切换-指定文件名：cmd+p
  - editor group
    - 切换-左右
    - 创建-左右
    - 创建-上下
  - 切换至上一个使用过的编辑器：cmd+alt+left/right
  - 折叠节点内容：cmd+k, cmd+1/2/3/4
  - 展开节点内容：cmd+k，cmd+j
- 编辑
  - 移动光标
    - 移动光标-单个字符：alt+left/right 
    - 移动光标-到括号末尾：cmd+shift+\
    - 移动光标-到文档首、末尾：cmd+up/down
    - 移动光标-到括号外：对应符号右侧
  - 移动文本
    - 移动行-上下移动：alt+top/down
    - 移动多行-上下移动：
    - 选择文本-单个字符：
    - 选择文本-多个连续字符：
    - 选择文本-单行：
  - 同时编辑多处：cmd+alt+方向键
  - 新建行：shift+cmd+enter
  - 粘贴图片：ctrl+alt+v
- 查找
  - 查找光标所在单词：cmd+e

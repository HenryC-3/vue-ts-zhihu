- 3-14 Teleport - 瞬间移动 第一部分
	- 知识点
		- teleport 是为了解决什么问题？
			- 以常用组件 Dialog 为例，Dialog 作为一个通用组件，可能会被各种组件调用，最终导致 Dialog 组件嵌套层级过深
			![](attachments/组件深度嵌套.png)
		- 组件嵌套过深会带来什么问题？
			- Dialog 的样式容易被干扰
			- 样式在其他组件中，容易非常混乱
		- teleport 做了什么？
			- teleport 将组件渲染到指定位置，跳出了层级，避免了样式干扰问题，但依旧保留了书写形式上的组件嵌套关系
